# 生日倒计时网页应用使用说明

## 功能介绍

本生日倒计时网页应用是一个用户友好的工具，支持多用户管理，实时追踪距离用户生日的剩余时间，精确到秒级，并提供视觉吸引力强的动态背景效果和翻页动画。

## 核心功能

- **多用户管理**：支持创建、切换、删除多个用户档案
- **用户名称输入**：强制要求输入用户名称，确保用户信息完整
- **生日日期输入**：支持通过日期选择器或手动输入设置生日日期
- **实时倒计时**：精确到秒级的倒计时显示，格式为"X天X时X分X秒"
- **翻页动画**：当时间单位变化时呈现平滑的向上翻书页动画效果
- **动态背景**：五彩缤纷的渐变色彩流动效果配合粒子动画
- **本地存储**：自动保存所有用户信息，刷新页面后保持设置
- **响应式设计**：在不同屏幕尺寸下保持良好的显示效果

## 如何使用

### 创建新用户
1. 打开 `index.html` 文件在浏览器中
2. 在"用户名称"输入框中输入用户名称（必填）
3. 在"手动输入生日日期"输入框中选择或输入生日日期（必填）
4. 点击"创建用户"按钮
5. 系统会自动将该用户设为当前用户，并开始倒计时

### 切换用户
1. 在页面底部的"用户列表"区域查看所有已创建的用户
2. 点击任意用户卡片即可切换到该用户
3. 切换后，倒计时会自动更新为该用户的生日倒计时
4. 被选中的用户卡片会有明显的视觉高亮效果

### 更新用户信息
1. 点击要更新的用户卡片
2. 输入框会自动填充该用户的信息
3. 修改用户名称或生日日期
4. 点击"更新信息"按钮保存修改

### 删除用户
1. 将鼠标悬停在要删除的用户卡片上
2. 点击卡片右上角的"×"按钮
3. 确认删除操作

## 视觉效果

- **当前用户显示**：顶部显示当前选中的用户名称
- **数字卡片**：横向排列的倒计时数字卡片，清晰醒目
- **翻页动画**：数字变化时呈现向上翻书页的流畅动画效果
- **背景效果**：深色渐变背景配合浮动粒子动画，营造梦幻氛围
- **时间分隔符**：闪烁的冒号分隔符，增强视觉效果
- **用户卡片**：卡片式用户展示，选中状态有明显的边框高亮和背景变化

## 技术实现

- **前端**：HTML5, CSS3, JavaScript
- **动画**：使用CSS3 3D变换实现翻页效果
- **背景**：使用CSS渐变和JavaScript粒子动画实现动态效果
- **存储**：使用localStorage存储所有用户信息和当前选中用户

## 响应式设计

应用采用响应式设计，在不同设备上均有良好的显示效果：
- **桌面端**：完整显示所有元素，大尺寸数字卡片
- **平板**：自动调整元素大小和间距
- **手机**：垂直排列输入区域，减小倒计时数字大小

## 浏览器兼容性

支持所有现代浏览器，包括：
- Chrome
- Firefox
- Safari
- Edge

## 数据存储说明

- 所有用户信息存储在浏览器的localStorage中
- 存储键名：`birthdayUsers`（用户列表）、`currentUser`（当前选中用户ID）
- 数据格式：JSON格式，包含用户ID、名称和生日日期
- 清除浏览器数据会导致用户信息丢失

## 注意事项

- 创建用户时必须输入用户名称和生日日期
- 用户名称不能为空
- 生日日期格式为标准日期格式（YYYY-MM-DD）
- 应用会自动计算下一个生日，即使当前日期已过今年的生日
- 本地存储功能依赖于浏览器的localStorage支持
- 粒子动画数量已优化，不会影响页面性能
