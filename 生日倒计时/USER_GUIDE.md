## 1. 应用简介

### 1.1 应用概述

生日倒计时网页应用是一个功能完整的生日管理工具，支持多用户管理、实时倒计时显示、农历日期转换等功能。应用采用现代化的界面设计，提供流畅的用户体验。

### 1.2 核心功能

- **多用户管理**：支持创建、切换、删除多个用户档案
- **实时倒计时**：精确到秒级的倒计时显示
- **翻页动画**：数字变化时的流畅动画效果
- **农历支持**：自动显示农历日期
- **本地存储**：数据自动保存，刷新不丢失

### 1.3 系统要求

| 项目 | 要求 |
|-----|------|
| 操作系统 | Windows / macOS / Linux / iOS / Android |
| 浏览器 | Chrome 60+ / Firefox 55+ / Safari 12+ / Edge 79+ |
| 网络 | 首次加载需要网络，之后可离线使用 |
| 存储 | 约 5MB 浏览器存储空间 |

## 3. 功能模块详细操作步骤

### 3.1 创建新用户

#### 步骤说明

1. **输入用户名称**
   - 在"用户名称"输入框中输入名称
   - 名称不能为空

2. **选择生日日期**
   
   **方式一：使用日期选择器**
   - 点击日期输入框
   - 在弹出的日历中选择日期
   
   **方式二：手动输入**
   - 点击"手动输入"按钮切换模式
   - 按照YYYY-MM-DD格式输入日期
   - 例如：1990-05-15

3. **点击"创建用户"按钮**
   - 用户创建成功后自动显示倒计时

![创建用户流程](screenshots/03-create-user.png)

#### 输入验证

| 验证项 | 规则 | 错误提示 |
|-------|------|---------|
| 用户名称 | 不能为空 | "请输入用户名称！" |
| 日期格式 | YYYY-MM-DD | "日期格式不正确" |
| 年份范围 | 1900-2100 | "年份应在1900-2100之间" |
| 月份范围 | 01-12 | "月份应在01-12之间" |
| 日期范围 | 根据月份 | "该月份日期应在01-XX之间" |

## 4. 常见问题解答(FAQ)

### Q1: 数据会丢失吗？

**A:** 数据存储在浏览器的localStorage中，只要不清除浏览器数据，数据会一直保留。建议定期备份重要信息。

### Q2: 支持哪些浏览器？

**A:** 支持所有现代浏览器：
- Chrome 60及以上版本
- Firefox 55及以上版本
- Safari 12及以上版本
- Edge 79及以上版本

### Q3: 为什么农历日期显示不正确？

**A:** 请检查以下几点：
1. 确保输入的公历日期正确
2. 农历转换支持1900-2100年的日期范围
3. 如果日期超出范围，将无法正确转换

### Q4: 如何在多台设备间同步数据？

**A:** 当前版本不支持自动同步。如需在多台设备使用，需要手动创建相同的用户信息。

### Q5: 为什么刷新页面后数据还在？

**A:** 应用使用浏览器的localStorage存储数据，刷新页面不会清除存储的数据。

### Q6: 可以设置生日时间吗？

**A:** 当前版本仅支持设置生日日期，默认从当天开始计算倒计时。

### Q7: 如何清除所有数据？

**A:** 可以通过以下方式清除：
1. 打开浏览器开发者工具（F12）
2. 进入"Application"或"存储"选项卡
3. 找到"Local Storage"
4. 删除相关存储项

## 5. 故障排除方法

### 5.1 页面无法加载

#### 症状
- 页面显示空白
- 样式错乱
- 功能无法使用

#### 解决方法

1. **检查文件完整性**
   - 确保所有文件都已正确下载
   - 检查文件是否在正确位置

2. **清除浏览器缓存**
   - 按 Ctrl+Shift+Delete（Windows）或 Cmd+Shift+Delete（Mac）
   - 选择清除缓存和Cookie

3. **尝试其他浏览器**
   - 换用Chrome或Firefox浏览器

### 5.2 倒计时不更新

#### 症状
- 倒计时数字静止不动
- 时间显示不正确

#### 解决方法

1. **检查系统时间**
   - 确保电脑系统时间正确
   - 检查时区设置

2. **刷新页面**
   - 按 F5 或 Ctrl+R 刷新页面

3. **检查浏览器控制台**
   - 按 F12 打开开发者工具
   - 查看"Console"选项卡是否有错误

### 5.3 用户数据丢失

#### 症状
- 刷新后用户列表为空
- 之前创建的用户不见了

#### 解决方法

1. **检查localStorage**
   - 打开开发者工具（F12）
   - 进入"Application" → "Local Storage"
   - 查看是否有"birthdayUsers"项

2. **检查浏览器设置**
   - 确保未开启"退出时清除浏览数据"
   - 检查是否有隐私保护软件清除数据

### 5.4 农历日期显示异常

#### 症状
- 农历日期为空
- 农历日期明显错误

#### 解决方法

1. **检查日期范围**
   - 确保日期在1900-2100年之间

2. **检查日期格式**
   - 确保公历日期格式正确

3. **刷新页面**
   - 重新加载农历转换模块

### 5.5 动画卡顿

#### 症状
- 翻页动画不流畅
- 页面响应缓慢

#### 解决方法

1. **关闭其他标签页**
   - 减少浏览器内存占用

2. **检查CPU使用率**
   - 关闭占用CPU的其他程序

3. **更新浏览器**
   - 使用最新版本的浏览器
